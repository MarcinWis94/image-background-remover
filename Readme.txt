# Image Background Remover

A Python script to remove backgrounds from images in a folder and save the processed images in another folder.

## Features
- Processes all images in a given folder.
- Saves processed images to an output folder.
- Supports `.png`, `.jpg`, and `.jpeg` formats.

## How to Use
1. Install dependencies: `pip install rembg`.
2. Set the input and output folder paths in the script.
3. Run the script with `python background_remover.py`.
